Admin — Run Scripts

Utility scripts for maintaining Photos & Thumbnails.

First upload your photos to the /WWW/MEDIA linked directory, then use these:

  • Run 'Generate Cache' — Generate & cache web-friendly photos & thumbnails for pictures in the /WWW/MEDIA folder. Then runs 'Cull Orphans'.
  • Run 'Cull Orphans' — Cull any /WWW/CACHE photos & thumbnails that aren't in /WWW/MEDIA anymore. Then runs 'Compile List'.
  • Run 'Gallery List' — Compile the list of Gallery photos to display from all /WWW/CACHE photos.
  • See Status — See status output of current process.
  • See Cached Media List — JSON array of files displayed by the Gallery produced by 'Compile List'.
  • See Env Vars — See env vars that get passed to apache.
  • Run 'Nuke' — Delete all photos & thumbs in /WWW/CACHE (original media files will remain untouched). 'Generate Cache' will have to be re-run.

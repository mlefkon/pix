Admin — Run Scripts

Utility scripts for maintaining Photos & Thumbnails.

First upload your photos to the /storagebox/pix/media/ directory, then use these:

  • Run 'Generate Web Cache' — Generates & caches web-friendly photos and thumbnails for any new pictures in the /media folder. Then runs 'Cull Orphans'.
  • Run 'Cull Orphans' — Culls any /CACHE photos & thumbs that aren't in /MEDIA anymore. Then runs 'Compile List'.
  • Run 'Compile List' — Generates the list of Gallery photos to display from all /CACHE photos.
  • See Status — See status of current process.
  • See Cached Media List — JSON array of files displayed by the Gallery.
  • See Env Vars — See env vars that get passed to apache.
  • Run 'Nuke' — Delete all Cached photos & thumbs (original media files will remain untouched)
